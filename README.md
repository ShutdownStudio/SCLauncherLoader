# Shutdown_craft Launcher Loader
 **⚠️警告：本软件无法在某些电脑运行，在Windows7运行似乎会遇到错误** 


#### 介绍
Shutdown_craft Launcher准备程序</br>
1. 检测 Minecraft运行环境 并自动安装
2. 检测 启动器运行环境 并自动安装
3. 下载最新版本的 启动器(整合包)


#### 使用说明

1. 在 Windows7SP1 实测无法运行
2. 全自动下载安装运行配置环境
3. 杀毒软件可能会报毒
4. 大部分代码均为原创(详情见特别感谢)
5. 可任意修改代码，但是请保留我们(工作室、团队、开发者三者之一)的联系信息、版权信息
